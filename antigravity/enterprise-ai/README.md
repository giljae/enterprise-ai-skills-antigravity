# Enterprise AI for Antigravity

Antigravity-native consulting and PM skills for structured thinking, executive communication, prioritization, meeting prep, deck workflows, and lightweight data/deck automation.

## Install

From the repository root:

```bash
./scripts/install_antigravity.sh
```

This installs the suite to:

```text
~/.gemini/antigravity/skills/enterprise-ai/
```

## Commands

- `/enterprise-ai` routes ambiguous business requests.
- `/scpr` structures strategic arguments.
- `/issue-tree` decomposes problems into MECE hypotheses.
- `/decision-memo` drafts 1-page yes/no memos.
- `/storyline` creates claim-based deck flows.
- `/prioritize` scores roadmap items.
- `/meeting-prep` prepares outcome-driven meetings.
- `/ai-use-case-score` ranks personal AI use cases.
- `/mckinsey-critic` reviews memos, decks, and strategies.
- `/deck-pipeline` runs a deck build/review/fix workflow.
- `/data-insights` analyzes CSV/Excel files.
- `/gamma-deck` creates Gamma decks from storylines.

## Design

Each command has a compact `SKILL.md` for fast loading. Full frameworks, examples, and rubrics live in `references/` and `examples/` and should be read only when needed.
